# java_practice_projects
# 1. Movie Rating Analyzer

## Project Description
This is your assessment project after learning **if statements**, **ArrayList**, **for loops with range**, and **methods**.

You will build a program that analyzes a list of movies and their ratings. 
All data is hardcoded — no Scanner is needed.

## Requirements

You must implement **5 methods**:

1. `printAllMovies(...)`  
2. `calculateAverageRating(...)`  
3. `countHighlyRated(...)`  
4. `printRatingCategories(...)`  
5. `findHighestAndLowestRated(...)`

### Rules
- Only use: `if` / `else if` / `else`, `ArrayList`, indexed `for` loops (`for (int i = 0; i < size; i++)`), and methods.
- All methods must be `static`.
- Do **not** use Scanner, enhanced for-each loop, or while loops.
- Add comments in your code to explain your logic.

## How to Run
1. Fork or use this template repository
2. Complete the TODOs in `MovieRatingAnalyzer.java`
3. Compile and run the program
4. Test with the given sample data
